# 乒乓球世界 - Table Tennis World

一个介绍乒乓球运动的多页面静态网站，包含乒乓球历史、规则、技术、装备、传奇运动员和重要赛事等内容。

## 📁 网站结构

```
tabletennis-site/
├── index.html          # 首页
├── history.html        # 乒乓球历史
├── rules.html          # 比赛规则
├── techniques.html     # 技术要领
├── equipment.html      # 运动装备
├── players.html        # 传奇运动员
├── tournaments.html    # 重要赛事
├── css/
│   └── styles.css      # 样式表
└── js/
    └── script.js       # 交互脚本
```

## 🌟 网站特色

### 页面内容
- **首页**: 快速导航、荣誉统计、运动员和赛事预览
- **历史**: 从 1880 年代起源到现代发展的完整时间线
- **规则**: 得分、发球、场地、击球等基本规则和犯规判罚
- **技术**: 握拍方式、进攻技术、防守技术、步法移动
- **装备**: 球拍、乒乓球、服装鞋类、配件及保养方法
- **运动员**: 中外传奇选手详细介绍（马龙、樊振东、邓亚萍、瓦尔德内尔等）
- **赛事**: 奥运会、世锦赛、世界杯等三大赛及其他重要赛事

### 技术特点
- 响应式设计，支持手机、平板、桌面设备
- 现代化渐变配色和卡片式布局
- 平滑滚动导航
- 滚动动画效果
- 数字计数动画
- 移动端汉堡菜单
- 回到顶部按钮

## 🚀 使用方法

直接在浏览器中打开 `index.html` 文件即可浏览网站。

```bash
# 或者使用本地服务器
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# 然后访问 http://localhost:8000
```
